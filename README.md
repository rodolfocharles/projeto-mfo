Desenvolvi uma aplicação web de planejamento financeiro de longo prazo, focada em auxiliar profissionais como analistas e planejadores financeiros.
A ferramenta permite gerenciar o patrimônio do cliente, cadastrar movimentações recorrentes, criar simulações com diferentes cenários e gerar projeções mês a mês do patrimônio.
Além disso, a aplicação oferece comparação entre simulações, permitindo avaliar qual estratégia financeira é mais vantajosa no longo prazo.
